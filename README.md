# Lab
This git repository is for testing and experimenting with various tools and coding projects.

It also makes use of a development container to ensure that no required tools need to be installed locally.

## Infrastructure as Code (IaC)
This directory is used for testing out tools such as Pulumi, Terraform, and other tools to configure Infrastructure as code for deployments.

## Kubernetes
This directory is for testing basic Kubernetes code for basic deployments, Helm, and other Kubernetes tools.
